# 欢迎页实时同步工具 (Welcome Page Sync)

这是一个基于纯前端技术实现的“实时同步”欢迎页。无需后端服务器，利用 URL Hash 技术传递状态。

## 功能特点
1.  **实时内容同步**: 修改页面文字，URL 自动更新。分享链接或二维码后，对方看到的页面内容将与你一致。
2.  **二维码分享**: 一键生成当前状态的二维码，方便手机扫码访问。
3.  **零成本部署**: 完全基于 GitHub Pages，免费且稳定。
4.  **中国访问优化**: 核心依赖库本地化，确保国内访问速度。

## 使用指南
1.  **访问页面**: 打开部署后的 GitHub Pages 链接。
2.  **编辑文字**: 点击屏幕中央的“欢迎”文字，直接输入你想要展示的内容。
3.  **生成分享**: 点击“生成分享二维码”按钮。
4.  **扫码查看**: 让其他人扫描生成的二维码，他们将看到和你一样的文字内容。

## 部署说明 (Deployment)
本项目专为 GitHub Pages 设计。
1.  将代码 Push 到 GitHub 仓库。
2.  进入仓库 **Settings** -> **Pages**。
3.  在 **Build and deployment** 下，选择 **Source** 为 `Deploy from a branch`。
4.  选择 `main` 分支 (或 `master`) 并保存。
5.  等待几分钟，GitHub 会生成访问链接 (例如 `https://youngzy607-cpu.github.io/Test_20251229/`)。

## 技术原理
*   使用 `window.location.hash` 存储页面状态。
*   使用 `qrcode.js` 在客户端生成二维码。
