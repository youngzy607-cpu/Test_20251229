# Kelly Pool 分牌助手 (Pool Card Dealer)

一个基于 Web 的轻量级 Kelly Pool（凯利台球）分牌与游戏辅助工具。支持多人手机在线同步查看手牌，房主控制游戏进程。

## 🌟 特性
*   📱 **手机适配**：专为移动端设计的 UI，操作便捷。
*   🔄 **实时同步**：基于 MQTT 协议，所有玩家状态实时同步。
*   🔒 **隐私保护**：玩家只能看到自己的手牌，房主可控制游戏进程。
*   🚀 **无需安装**：网页版即开即用，支持扫码加入。
*   🇨🇳 **国内优化**：使用国内访问友好的 MQTT 服务器。

## 🎮 游戏规则 (Kelly Pool)
1.  **准备**: 所有人拿出手机，进入游戏房间。
2.  **发牌**: 房主点击“开始发牌”，系统随机将 1-15 号球分配给每位玩家。
3.  **游戏**: 
    *   在台球桌上正常击球。
    *   当某个号码的球（例如 5 号）被打进袋中。
    *   房主在手机上点击“5号球”。
    *   持有 5 号牌的玩家手机上，该牌会自动打叉/变灰。
4.  **获胜**: 第一位所有手牌都被打叉的玩家获胜（或根据具体规则决定输赢）。

## 🛠️ 技术实现
*   **前端**: 原生 HTML5 / CSS3 / JavaScript
*   **通信**: MQTT over WebSocket (使用 `broker.emqx.io` 公共服务)
*   **架构**: Host-Client 模式，房主端作为游戏主机维护状态。

## 🚀 部署与使用
本项目托管于 GitHub Pages，访问即可使用。
1.  打开首页。
2.  **房主**: 点击“我是房主”，创建房间号。
3.  **玩家**: 输入相同的房间号，点击“加入房间”。
