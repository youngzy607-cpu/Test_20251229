# Kelly Pool 分牌助手 (Pool Card Dealer)

一个基于 Web 的轻量级分牌与游戏辅助工具。支持多人手机在线同步查看手牌，房主控制游戏进程。
目前已升级支持 **54张扑克牌** 模式，适用于各类需要分牌的游戏。

## 🌟 特性
*   📱 **手机适配**：专为移动端设计的 UI，操作便捷。
*   🔄 **实时同步**：基于 MQTT 协议，所有玩家状态实时同步。
*   🔒 **隐私保护**：
    *   **玩家视角**：玩家只能看到自己的手牌。
    *   **房主视角**：房主作为玩家参与游戏，会分到手牌。房主控制台**不再显示其他玩家的手牌**，只显示连接状态，确保公平。
*   🃏 **扑克模式**：支持标准54张扑克牌（含大小王），支持自动洗牌与排序（从小到大）。
*   🚀 **无需安装**：网页版即开即用，支持扫码加入。
*   🇨🇳 **国内优化**：使用国内访问友好的 MQTT 服务器。

## 🎮 游戏规则 (通用分牌)
1.  **准备**: 所有人拿出手机，进入游戏房间。
2.  **发牌**: 
    *   房主设置每人手牌数量（或自动平分）。
    *   房主也会被视为玩家之一，参与分牌。
    *   点击“开始发牌”。
3.  **显示**: 
    *   系统使用标准54张扑克牌进行随机洗牌。
    *   玩家（包括房主）手机上会显示分到的手牌，并自动按从小到大排序。
4.  **互动**: 
    *   房主可点击控制台上的扑克牌标记为“已打出”。
    *   持有该牌的玩家手机上，对应卡牌会显示已打出状态。

## 🛠️ 技术实现
*   **前端**: 原生 HTML5 / CSS3 / JavaScript
*   **通信**: MQTT over WebSocket (使用 `broker.emqx.io` 公共服务)
*   **架构**: Host-Client 模式，房主端作为游戏主机维护状态。

## 🚀 部署与使用
本项目托管于 GitHub Pages，访问即可使用。
1.  打开首页。
2.  **房主**: 点击“我是房主”，创建房间号。
3.  **玩家**: 输入相同的房间号，点击“加入房间”。
